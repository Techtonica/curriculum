What is React state?
- [ ] the properties of an object
- [ ] a callback used to obtain props
- [ ] a built-in React object that is used to contain data or information about the component
- [ ] a built-in React element that collects information from a hook
 
How do you pass data from a child to a parent component
- [ ] Create a callback function in the parent, which will retrieve the data from the child. Pass the callback function to the child as props from the parent. The child calls the parent callback using props and passes the data to the parent.
 - [ ] You can use props to transfer the data, the child will access the data in the props.
- [ ] When sharing data from parent to child component, we use props. Props data is shared by the parent and child cannot be changed as they are read-only
- [ ] Every parent component can pass some information to its child components by giving them props.

What is the difference between React Props and State?
- [ ] Props are mutable and can be changed within a component, while state is immutable and cannot be updated function.
- [ ] Props are immutable and can be changed within a component, while state is mutable and can't be updated using the setState function.
- [ ] Props are used to pass data from parent to child; while state is used to manage data within a component.
