What is the difference between unit testing and integration testing?
- [ ] Unit tests will check all your code in a specific component only, while integration testing will check for all components
- [ ] In unit testing, a small piece of code is tested. In integration testing, the test coverage and connection between code is tested.
- [ ] Unit testing is one test at a time, integration testing is more than one test at a time

- [ ] transform: <duration> <easing-function> <delay>;
- [ ] transition: <duration> <easing-function> <delay>;
- [ ] transition: <transformation> <easing-function> <delay>;
