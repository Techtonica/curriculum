# Arrays

An array is a data structure that stores an ordered collection of values and lets you access elements by index.

Even if you primarily work in JavaScript (where arrays are dynamic and can behave like lists), thinking about “array operations” and their tradeoffs is a core skill for writing efficient code.

## Prerequisites

- [Runtime Complexity](/runtime-complexity/runtime-complexity.md)

## In This Curriculum

## What to Focus On

- Access by index vs searching
- Insertion/removal cost (especially at the beginning or middle)
- When to use arrays vs linked lists, maps, or sets
